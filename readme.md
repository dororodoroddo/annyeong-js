# annyeong-js *신버전*
## Introduction
학습용 javascript/typescript로 작성된 SPA 라이브러리

### 특징
* dom element의 속성값과 상태의 변화를 반응형으로 연결하여 비교없이 변화가 있는 부분만 변화시켜 비교 시간을 절약할 수 있습니다.
* .fullSize(), .flexRow() 와 같이 이터러블 프로토콜과 같은 형식으로 스타일을 지정할 수 있다.
* frame단위 렌더링을 통한 최적화 및 활용
